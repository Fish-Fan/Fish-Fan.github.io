文件命名

  • 源代码的文件名必须和 公共类 的名字相同
  • 一个Java文件只能有一个 public类
  • 如果文件不止一个类，且没有 public类，那么Java文件命名没有限制

Java数据类型

  1. 4种整型
  • long 8byte 贼长
  • int 4byte -20亿~20亿
  • short 2byte -3万~3万
  • byte 1byte -128~127

Java程序必须保证在所有机器上都能运行一致，所以各种数据类型的取值必须固定。
C和C++的类型大小和平台有关。

  1. 两种浮点
  • float 4byte 6~7位
  • double 8byte 15位

三个特殊的浮点数值:

  • 正无穷大 -> Double.POSITIVE_INFINITY
  • 负无穷大 -> Doubel.NEGATIVE_INFINITY
  • NaN -> Double.NEGATIVE_INFINITY
