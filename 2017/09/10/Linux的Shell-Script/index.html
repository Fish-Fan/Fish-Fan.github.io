注意事项 :

  1. 命令，参数间的多个空白都会被忽略掉
  2. 空白行也会被忽略掉，[tab]所得空白等于[space]所得空白
  3. 如果得到一个Enter符号(CR),就尝试执行该命令
  4. “#”号后面为注释

执行方法 :

  1. 直接命令执行(.sh文件必须具有rx权限)

  2. 以bash进程来执行(有r权限即可执行),该种执行方式在子进程中执行，所以声明的变量在父进程中访问不到

  3. 用source执行,该种执行方式是使用父进程执行，故声明的变量可以访问
